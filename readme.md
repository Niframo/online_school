# Система управления онлайн-школой v2.1

Консольное приложение на Python для управления учебным процессом. Позволяет вести учет студентов, преподавателей и курсов, выставлять оценки за домашние задания, собирать аналитику и визуализировать успеваемость.

## Основной функционал

* **Управление пользователями:** Добавление студентов и преподавателей, привязка их к курсам.
* **Структура курсов:** Поддержка сложной структуры (Курс ➔ Модули ➔ Уроки).
* **Система оценивания:** Выставление оценок за домашние задания в конкретных уроках и автоматический расчет итогового балла при завершении курса.
* **Постоянное хранение данных:** Автоматическое сохранение и загрузка состояния системы из файла `school_data.json`. При первом запуске база автоматически заполняется тестовыми данными (seed data).
* **Аналитика и отчетность:**
  * Формирование детализированных отчетов по преподавателям (активные/завершенные курсы, средняя успеваемость).
  * Экспорт отчетов по студентам в формат `.csv`.
  * **Визуализация:** Построение графиков успеваемости студентов с помощью библиотеки `matplotlib`.

## Требования

* **Python 3.7 или выше**

* **Библиотека matplotlib (для отрисовки графиков успеваемости)**

## Установка

1. Скопируйте проект в директорию
2. Установите зависимости
```bash
pip install -r requirements.txt
```
3. Запустите файл `main.py`

## Запуск тестов
Чтобы запустить проверку логики вашего проекта, выполните в терминале команду:
```bash
python tests.py
```
